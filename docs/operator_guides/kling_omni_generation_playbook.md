# Kling Omni Generation Playbook

Kling Omni generation is not active in Batch 5.85. This guide is a placeholder
for Phase 3 and must not be treated as an executable workflow until Batch 8.

## Activation Gate

Do not generate Kling Omni clips until all of the following are true:

- Batch 8 is implemented.
- Required element packs are locked by human PR.
- Storyboard direction is selected by the approved workflow.
- Scene-level Omni prompt and shot guidance are available.
- Kling model guidance is active and recently reviewed.
- Video take review and clip locking workflows exist.

## Current Safe Action

If operator guidance points here before Batch 8, treat the task as blocked and
return to the current Batch 5.85 guidance output.

## Storage Safety

Do not commit `.mp4`, `.mov`, or other video binaries in this phase. Use the
future approved external storage reference workflow.
