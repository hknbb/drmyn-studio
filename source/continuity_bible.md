## Title and scope note

This document defines the continuity law for `CLOSING PRICE` using screenplay-grounded source evidence. It governs downstream scene work, planning records, prompts, and later visual production. It does not replace scene-by-scene continuity tracking; it sets the rules those downstream records must obey.

## Continuity philosophy

- Continuity in this project is a consequence system, not a cosmetics check.
- If an object, wound, route, lock, timing window, or knowledge gain matters once, it usually matters again.
- The screenplay's strongest continuity engine is accumulation: surveillance pressure, bodily degradation, route logic, and operational knowledge all carry forward.
- When evidence is thin, preserve the uncertainty rather than normalizing it into false certainty.

## Spatial continuity rules

- Spaces in this film are operational. Geography must remain usable.
- Service entrances, loading bays, drainage conduits, checkpoints, outer gates, wall walks, windows, and corridor angles are not background dressing.
- A trap may include real geography. Do not flatten partially real space into either "fully safe" or "fully fake" when the source describes both.
- If Nadia verifies a route visually, later scene work must respect that route logic unless the script states a change.
- Compound and institutional spaces should preserve layered access behavior: outer perimeter, checkpoint, corridor, control room, antechamber, server core.

- If a later sequence adds a second serious wound, both the old and new impairments must be considered together.
